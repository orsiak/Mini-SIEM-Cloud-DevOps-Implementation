Mini SIEM 
This project is for learning purposes in DevOps, cloud, and web app technologies.
The first step is to build a local mini web logging app to generate logs needed for mini-siem. There will also be other log sources as services like SSH. These logs are then sent to a SIEM system, implemented using the ELK Stack and Grafana.
The second step is an AWS cloud DevOps-style implementation, demonstrating a full DevOps workflow:
-Kubernetes for orchestration
-Terraform for infrastructure as code
-GitHub Actions for CI/CD
-AWS cloud deployment

Stack:
-Flask
-ELK Stack
-Grafana
-Docker
-Kubernetes
-AWS Cloud
-Terraform
-GitHub Actions
